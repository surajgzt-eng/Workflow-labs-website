# ViralVibe Labs E-Commerce Storefront Website

A visually satisfying, highly responsive, and interactive single-page e-commerce landing page built for **ViralVibe Labs**.

## Features

- **Premium Dark Aesthetics**: Styled in electric violet (`#7C3AED`) and deep charcoal with modern typography and neon effects.
- **Fast Dynamic Liquid Light**: Features an interactive, high-performance cursor-following glowing orb that responds to mouse movement and velocity with fluid physics.
- **Viral Product Catalog**: Displays five curated daily problem-solvers with descriptions, tags, and pricing.
- **Interactive State-Based Cart Drawer**: A fully functional Ajax-simulated shopping basket. Users can add products, open/close the sliding drawer, change item quantities, and watch the subtotal update dynamically.
- **Responsive Accordion FAQs**: Seamlessly integrated question and answer section with fluid CSS transition toggles.
- **Newsletter Subscription**: Responsive registration form designed to capture user emails for high-impact retargeting.

## File Structure

- `index.html` - Complete website template combining structures, styles, script logic, and cursor-following canvas layers.
- `README.md` - Repository information and documentation.

## How to Deploy (GitHub Pages)

To launch this website live on GitHub Pages for free:
1. Go to the settings of this repository on GitHub.
2. Under **Pages**, select the `master` branch as the build source and click **Save**.
3. Within minutes, GitHub will host your storefront live at `https://yourusername.github.io/Workflow-labs-website/`.
