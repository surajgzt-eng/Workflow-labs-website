# Workflow Labs Website

AI-Powered Creator Tools landing page built for Workflow Labs startup.

## Features

- Modern dark theme with neon gradients
- Interactive cursor-following liquid light effect
- Responsive design for all devices
- Smooth animations and transitions

## Tech Stack

- Pure HTML/CSS/JavaScript (no framework)
- Google Fonts (Inter)
- CSS animations and effects

## Getting Started

Simply open `index.html` in a browser to view the website.
